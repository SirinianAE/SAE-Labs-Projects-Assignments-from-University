Για να μπορέσει κανείς να τρέξει το πρόγραμμα AnimeCalendar για android συσκευές είναι αναγκαίο να έχει εγκατεστημένο το πρόγραμμα Android Studio. 
Η εγκατάσταση του Android Studio ξεκινάει από το site: https://developer.android.com/studio/index.html όπου και βρίσκετε το αντίστοιχο .exe αρχείο του.
Αφού εγκατασταθεί το πρόγραμμα με επιτυχία ψάξτε να βρείτε έναν φάκελο που ονομάζεται AndroidStudioProjects το οποίο συνήθως βρίσκεται στον φάκελο Admin (για υπολογιστές με Windows λειτουργικό σύστημα). 
Αφού βρείτε τον φάκελο AndroidStudioProjects αντιγράψτε τον φάκελο AnimeCalendar από το dvd σ ’αυτό. 
Αφού αντιγράψετε το AnimeCalendar στο AndroidStudioProjects μπορείτε να ανοίξετε το AndroidStudioProject.

Αν το android studio ήταν εγκατεστημένο είδη στον υπολογιστή υπάρχει μεγάλη δυνατότητα να ανοίξει απευθείας σε κάποιο project, σ ‘αυτήν την περίπτωση κάντε File>Close Project. 
Κάντε Open an existing Android Studio project από το αρχικό μενού του android studio και επιλέξτε τον φάκελο AnimeCalendar για το οποίο αναφερθήκαμε παραπάνω. 

Ανοίξτε το AVD Manager και επιλέξτε την συσκευή που θέλετε να τρέξετε το πρόγραμμα πχ. Nexus S. 
Αν δεν υπάρχει είδη κάποια συσκευή κάντε Create Virtual Device…>Clone Device…>Finish (ενδιάμεσα μπορεί να χρειαστεί να αλλάξετε κάτι). 
Έπειτα κάντε run ‘app’ διαλέξτε μια συσκευή και πατήστε ok. 
Το πρόγραμμα θα ανοίξει αργά η γρήγορα ανάλογα με την ταχύτατα του υπολογιστή σας (το android studio είναι ένα βαρύ πρόγραμμα και χεριάζετε γενικά πολύ RAM και καλό επεξεργαστή).

Το πρόγραμμα μπορείτε να δοκιμάσετε να το τρέξετε και σε μία φυσική συσκευή αν το επιθυμείτε αλλά επειδή οι συσκευές με android λειτουργικό είναι πολλές θα χρειαστεί να κάνετε μια έρευνα από μόνοι σας.
